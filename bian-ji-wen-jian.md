# **编辑文件**

**vi** 可以在命令行下编辑文件。

```
vi 要编辑的文件路径
```

**练习：编辑文件**

```
vi ninghao-project/README.md
```

这样会打开要编辑的文件，使用方向键可以移动光标。

**编辑**

想要编辑按一下小 `i` ，这样会进入到 `vi` 编辑器的编辑模式。这时你可以编辑文件里的内容，修改完成以后，按 `esc` 可以退出编辑模式。

**搜索**

搜索文件里的内容可以按 `/` ，然后输入要搜索的关键词，`n` 继续查找，`N` 往前查找。

**保存**

输入 `:wq` ，可以保存对文件的修改。退出不保存文件，输入 `:q!` 。
